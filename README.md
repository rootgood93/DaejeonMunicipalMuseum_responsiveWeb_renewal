# 대전시립박물관 리뉴얼
메인 페이지를 새롭게 디자인하여 반응형으로 퍼블리싱하였습니다. 제이쿼리 플러그인을 사용하지 않고 직접 짜서 슬라이드와 메뉴바, 탭을 구현했습니다.
기획부터 퍼블리싱까지 전부 개인 작업으로 이루어졌습니다.
